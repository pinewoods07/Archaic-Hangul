# 옛한글 변환기 🖋️

현대 한글을 훈민정음 시대(15세기)의 **옛한글**로 변환해주는 웹사이트입니다.  
GitHub Pages로 무료 배포됩니다.

---

## ✨ 주요 기능

| 기능 | 설명 |
|------|------|
| **단어 사전** | 역사적으로 검증된 80여 개 단어를 정확하게 변환 |
| **아래아(ㆍ) 규칙** | 특정 초성 + ㅏ → ㆍ 자동 변환 |
| **구개음화 역적용** | 옛 한글의 ㄷ/ㅌ 형태 복원 |
| **변환 과정 설명** | 단어별 적용 규칙을 버튼 클릭으로 확인 |
| **복사 기능** | 변환 결과를 클립보드에 복사 |

## 🚀 GitHub Pages 배포 방법

### 1단계 — 저장소 생성
```bash
# GitHub에서 새 저장소 생성 후
git clone https://github.com/[내아이디]/[저장소이름].git
cd [저장소이름]
```

### 2단계 — 파일 복사 & 업로드
이 프로젝트의 모든 파일을 복사한 뒤:
```bash
git add .
git commit -m "✨ 옛한글 변환기 초기 배포"
git push origin main
```

### 3단계 — GitHub Pages 활성화
1. GitHub 저장소 → **Settings** 탭
2. 왼쪽 메뉴 **Pages** 클릭
3. **Source** → `Deploy from a branch`
4. **Branch** → `main` / `/ (root)` 선택 후 **Save**
5. 잠시 후 `https://[내아이디].github.io/[저장소이름]/` 에서 확인

